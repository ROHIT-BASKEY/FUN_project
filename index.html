Love Percentage

How It Works:

1. Combining Names with "LOVE": First, the code takes two names provided by the user. It puts these names together with the word "LOVE" in between. For example, if the names are "John" and "Jane", it would create "JOHNLOVEJANE".

2. Letters: The code then goes through each letter in "JOHNLOVEJANE" and counts how many times each letter appears. For instance, J appears twice, O once, H once, N twice, L once, V once, and E twice.

3. Saving Unique Letters: As it counts, the code also makes sure each letter is only counted once. It does this by storing each letter it has counted in a special list. If it comes across a letter that's already in this list, it skips counting it again.

4. Calculating with Numbers: With the counts of each letter (like 2 for J, 1 for O), the code then performs a series of calculations:
It starts by looking at the numbers from both ends towards the middle, adding them together, and keeping only the last digit of their sum. For example, if one end has 2 and the other end has 2, it adds them to get 4. If the sum is two digits (like 10), it only keeps the last digit (0).

5. Final Percentage: The last two numbers are then put together to form a percentage. If those numbers are, say, 3 and 5, the final "love percentage" would be 35%.

6. Displaying the Result: This percentage is then shown to the user, typically on a webpage where the names were entered.
